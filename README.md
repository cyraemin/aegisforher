# AEGIS FOR HER 🛡️
A global safety research project dedicated to gathering insights on personal security in transit and digital spaces.

## 🚀 Vision
AEGIS FOR HER is a data-driven safety research initiative designed to bridge the gap between individual safety perceptions and actionable community insights. By leveraging real-time, anonymous survey data, this platform seeks to highlight the lived experiences of individuals navigating transit and digital spaces. Our goal is to transform these observations into a foundation for safer infrastructure, better policy advocacy, and more inclusive technological safety features.

## 🛠️ Tech Stack
* **Frontend**: React.js, Vite, Tailwind CSS
* **Visualization**: Recharts (for analytical data dashboards)
* **Backend/Database**: Supabase
* **Deployment**: Vercel

## ⚙️ Setup Instructions
1. Clone the repo: `git clone https://github.com/cyraemin/project-aegis-for-her.git`
2. Install dependencies: `npm install`
3. Create a `.env` file with your Supabase credentials:
   - `VITE_SUPABASE_URL=your_url`
   - `VITE_SUPABASE_ANON_KEY=your_key`
4. Run locally: `npm run dev`
